Java Programming-oop-UCSD Setup Instruction

Setup for Windows host

  1. JDK: Oracle 8
  2. Eclipse latest version

Setup for Ubuntu

  1. #setup proxy: Network proxy GUI: ...
  2. setup JDK
    1. $ sudo -E add-apt-repository ppa:webupd8team/java //-E option to use system proxy if you have one.
    2. $ sudo apt-get update
    3. $ sudo apt-get install oracle-java8-installer
  3. setup eclipse
    1. $ sudo apt-cache search eclipse
    2. $ sudo apt-get install eclipse

# may need to do below step to remove the previous errors ## sudo rm /var/lib/apt/lists/lock ## sudo kill apt-PID

Setup common part: Configure eclipse http proxy server

  1. "Window", Preference", "General>Network Connection": Window proxy-networking, manual: child-prc.intel.com 914
  2. "Window", Preference", "JAVA", JRE RUNTIME "-Dhttp.proxyHost=child-prc.intel.com -Dhttp.proxyPort=914
